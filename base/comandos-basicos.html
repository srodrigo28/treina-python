Domine a Linha de Comando

Para um programador, o terminal não é uma tela preta assustadora, é a ferramenta mais rápida do sistema. Vamos desmistificar as diferenças entre os três gigantes.

Linux (Bash)

O padrão da indústria para servidores (VPS) e desenvolvimento web. Essencial para Docker e Deploy.

PowerShell

O terminal moderno do Windows. Poderoso, orientado a objetos e compatível com muitos comandos Linux (aliases).

CMD (Prompt)

O legado do MS-DOS. Limitado, mas onipresente em máquinas Windows antigas. Útil para scripts `.bat` simples.

PASSO 2

Top 10 Comandos do Dia a Dia

Você usará estes comandos em 90% do seu tempo como programador.

1 Listar

Vê o que tem dentro da pasta atual.

ls -la Linux (Detalhado)
ls PowerShell

2 Navegar (CD)

Entra e sai de pastas. Igual em todos.

cd nome_pasta Entrar
cd .. Voltar

3 Criar Pasta

Organize seus projetos.

mkdir projeto_flask Universal

4 Criar Arquivo

Cria um arquivo vazio para começar.

touch app.py Linux
ni app.py PowerShell (New-Item)

5 Privilégio Admin

Executar comando como Administrador.

sudo apt update Linux
runas /user:Admin cmd Windows

6 Testar Rede

Verifica se um site ou IP está respondendo.

ping google.com Universal

7 Buscar Texto

Encontra uma palavra dentro de arquivos.

grep "erro" log.txt Linux
sls "erro" log.txt PowerShell (Select-String)

8 Histórico

O que eu digitei antes? (Use Seta Cima para navegar).

history Linux/PS
F7 CMD (Janela)

9 IP da Máquina

Descobrir seu endereço na rede.

ip addr Linux
ipconfig Windows

10 Onde estou?

Mostra o caminho completo da pasta atual.

pwd Linux / PowerShell
cd CMD (sem argumentos)
