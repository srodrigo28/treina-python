Fluxo Flask Minimalista

Aprenda a criar uma aplicação web Python do zero absoluto. Sem excessos, apenas o essencial para capturar dados e renderizar páginas.

Objetivo

Criar um formulário web funcional que recebe o nome do usuário e exibe uma mensagem de boas-vindas.

Tecnologias

  • Python 3+
  • Flask (Microframework)
  • Jinja2 (Templates)
PASSO 1

Preparando o Terreno

Criação da pasta do projeto e isolamento com ambiente virtual.

1. Criar Pasta e Venv

Abra seu terminal e execute os comandos abaixo para organizar o projeto.

Terminal 
# Criar e entrar na pasta
mkdir meu_projeto
cd meu_projeto

# Criar ambiente virtual
python -m venv venv

2. Ativar e Instalar

Ative o ambiente isolado e instale o Flask.

Terminal 
# Windows
venv\Scripts\activate

# Linux / Mac
source venv/bin/activate

# Instalar Flask
pip install flask
Dica: Você saberá que funcionou se aparecer (venv) no início da linha do terminal.
PASSO 2

O Cérebro da Aplicação

Criando o arquivo app.py com rotas e validações.

app.py 
from flask import Flask, render_template, request

app = Flask(__name__)

# ROTA PRINCIPAL: Renderiza e Processa
@app.route('/', methods=['GET', 'POST'])
def index():
    mensagem = ''
    
    # Se enviou o formulário
    if request.method == 'POST':
        nome = request.form.get('nome_usuario')
        
        # Validação simples
        if not nome:
            mensagem = "⚠️ Erro: Nome vazio!"
        else:
            mensagem = f"✅ Olá, {nome}!"

    return render_template('index.html', msg=mensagem)

if __name__ == '__main__':
    app.run(debug=True)

request.form.get

Captura o dado enviado pelo input HTML através do atributo 'name'.

render_template

Carrega o arquivo HTML e injeta variáveis (como 'msg') nele.

PASSO 3

O Rosto da Aplicação

O arquivo HTML precisa ficar dentro de uma pasta chamada templates.

meu_projeto / templates / index.html
templates/index.html 
<!-- Estrutura Simples -->
<!DOCTYPE html>
<html>
<body>
    <h2>Formulário Flask</h2>

    <!-- Lógica do Jinja2 para exibir msg se existir -->
    {% if msg %}
        <div style="background:#eee; padding:10px; margin-bottom:10px;">
            {{ msg }}
        </div>
    {% endif %}

    <!-- Formulário envia para a mesma rota '/' -->
    <form method="POST" action="/">
        <label>Nome:</label>
        <input type="text" name="nome_usuario" placeholder="Digite...">
        <button type="submit">Enviar</button>
    </form>
</body>
</html>
